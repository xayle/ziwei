## 〇、定位（一句话）

> **简约国风为骨，宋式美学为章法。**  
> 学 **秩序、留白、纸墨层级**；不学 **纹样、动效、山水装饰**。

与竞品/模板的分水岭：**我们是辑录工具，不是算命直播间，也不是游戏 HUD。**

## 一、风格光谱（我们站在哪）

```text
AI 国风模板          浮生目标              纯宋式复刻
(米黄+窗格+印章)  ←——●————————→      (博物馆展柜)
     ✗                  ✓                    ✗
```

| 象限 | 代表 | 学什么 | 不学什么 |
|------|------|--------|----------|
| **文献阅读** | [识典古籍](https://www.shidianguji.com) | 纸色层级、横排可读、图文对照、低门槛 | 竖排为主、抄品牌色 |

| **刻本秩序** | 宋刻本版式 · [印刷博物馆](http://www.kepu.net.cn/gb/civilization/printing/evolve/evl316_02.html) | 版心、行界、天头地脚、双行小注 | 繁缛版框角花 |

| **反参考** | 千年3 游戏 UI · 站酷文旅红金 · AI 水墨节气卡 | — | 卷轴动效、印章反馈、水墨底、多色分区 |

文献共识：[宋代美学 · 百度百科](https://baike.baidu.com/item/%E5%AE%8B%E4%BB%A3%E7%BE%8E%E5%AD%A6/67390408) — **雅、淡、简、韵**；**以白当黑**；**格物致知**。

## 二、三层美术架构

| 层 | 名称 | 职责 | 占比 |
|----|------|------|------|
| **L1 骨** | 简约国风 | 纸墨、 serif、品牌 LOGO、少色 | 视觉 70% |
| **L2 筋** | 宋式章法 | 8pt 网格、版心、行界、分层读法 | 结构 100% |
| **L3 点** | 品牌徽章 | 铜弧 + 墨字 + 朱线（[`LOGO v2`](./FUSHENG-BRAND-LOGO-v2.md)） | 视觉 ≤8% |

**轻量化国风原则**（摘自行业实践）：水墨/动效 **只出现在关键节点**（Logo、卷封、切换），主体区保持 **扁平可读** — 见 [轻量化水墨 UI](http://wjdq.c2c3.cn/list/650014.html)。

## 三、核心美术数据（冻结）

### 3.1 色彩 · 纸墨铜朱（chrome 仅五色）

| 角色 | Token | Hex | 用途 | 面积上限 |
|------|-------|-----|------|----------|
| 纸 | `--brand-paper` | `#f5f0e6` | `body`、shell 外缘 | ~40% |
| 白 | `--surface` | `#fffaf5` | 内容区、盘面底 | ~35% |
| 墨 | `--brand-ink` | `#1a1410` | 标题、正文、表内主信息 | 文字主色 |
| 雾 | `--brand-mist` | `#6b5d4f` | 眉批、辅文、表头 | — |
| 淡 | `--text-3` | `#9a8b7a` | 卷次、脚注、占位 | ≥12px 仅装饰 |
| 铜 | `--brand-gold` | `#b8894d` | 主链、当前卷、KPI 数字 | **≤8%** |
| 铜深 | `--brand-gold-dark` | `#8b5e34` | hover、14px 铜标签 | — |
| 朱 | `--brand-cinnabar` | `#8b3a2a` | 缺失/双轨 **单行左线** | **≤2%** |
| 界 | `--border` / `--border-md` | `#e5dcc8` / `#d4c4a8` | 行界、册线 | 线 only |

**纸面两级（硬性）**：`纸 #f5f0e6 → 白 #fffaf5`。禁止 `surface` 套 `surface-2` 再套卡片。

**五行色** `--wx-*`：**仅**干支/五行字形，不作 UI chrome。

### 3.2 间距 · 8pt 网格

| Token | px | 用途 |
|-------|-----|------|
| `--sp-2` | 8 | 控件内微距 |
| `--sp-4` | 16 | 相关元素（卷次↔卷名） |
| `--sp-6` | 24 | 模块内分组 |
| `--sp-8` | 32 | 模块间 |
| 扩展 | 48 / 64 / 96 | 章间呼吸、首屏分段 |

**留白率目标**（借鉴 Plaud / 宋画）：

| 页面类型 | 内容占比 | 空气占比 |
|----------|----------|----------|
| 品牌首页首屏 | ≤35% | ≥65% |
| 卷目/登记表 | ~45% | ~55% |
| 命盘盘面页 | ~50% | ~50% |
| Trust 脚注 | 单行 | 不扩块 |

**宏观/中观/微观**（8dp 体系）：边距 ≥32px · 无关模块 ≥24px · 按钮内边距 ≥16px。

### 3.3 版心与宽度

| 名称 | 值 | 用途 |
|------|-----|------|
| `--page-max-w` | 1120px | 全站版心 |
| 品牌窄版心 | 640px | 首页开卷、读法层 |
| 顶栏高 | 56px | shell |
| 版心比例参考 | 7:10 ~ 0.7 | 宋刻本版框（内容区宜偏高） |

### 3.4 字体

| 层级 | Token / 字体 | 字号 | 字距 |
|------|----------------|------|------|
| 显示 | `--font-display` · LXGW Neo ZhiSong | `--fs-3xl` 28px | 0.12–0.16em |
| 篇题 | 同上 | `--fs-2xl` 22px | 0.08em |
| 正文 | 同上 | `--fs-md` 14px | 0.04em |
| 眉批/卷次 | 同上 | `--fs-xs` 12px | 0.14–0.28em |
| 脚注/校勘 | 同上 | 11–12px | 0.1em |
| UI 控件 | `--font-ui` | `--fs-sm` 13px | 正常 |

**行高**：正文 1.75–2.0（辑录阅读）；表内 1.5。

### 3.5 线条与容器

| 元素 | 规格 |
|------|------|
| 行界 | 1px `var(--border)` 或 `--border-md` |
| 册线（section 左） | 3px `var(--brand-gold)` — 每屏最多 1 处 |
| 圆角 | `--radius-codex` 8px；**禁止** 16px 卡片圆角堆叠 |
| 阴影 | 全页最多 **1 处** `--shadow-seal`（卷封/封面） |
| 边框分割 | 优先行界；**禁止** 多层级 box-shadow 卡片 |

见 [`FUSHENG-BRAND-LOGO-v2.md`](./FUSHENG-BRAND-LOGO-v2.md)。

| 场景 | 资产 | 尺寸 |
|------|------|------|
| 导航 | `fusheng-mark.svg` | 40–44px |
| 首页首屏 | `fusheng-mark.svg` | 200–280px |

## 四、宋式章法（结构语法）

### 4.1 刻本 ↔ 界面映射

| 宋刻本 | 浮生 UI |
|--------|---------|
| 天头（书名、卷次） | 顶栏篇题 + 卷标 |
| 版框 | 内容白底单块（非嵌套） |
| 行界 | 六柱表 / 卷目表 / 方盘宫格 |
| 双行小字注 | Trust 脚注、藏干 11–12px |
| 鱼尾/折页 | 章节 1px 分隔（非色块） |

### 4.2 呼吸分段（固定顺序）

```text
眉批（1 行）
  ↓ 24–32px
名题 / 器
  ↓ 32–48px
一句读法
  ↓ 48–64px
主体（一器）
  ↓ 48px
动作（1 链或 1 按钮）
```

### 4.3 首屏预算（按路由）

| 路由 | 唯一主角 | 首屏允许 | 首屏禁止 |
|------|----------|----------|----------|
| `/` | 品牌 LOGO | 系列名、进入链 | 姓名、四柱、六卷全表、KPI |
| `/profile` | 档案名 + 补全 | 一行 meta | 六卷、Trust 块 |
| `/new/bazi` | 六柱界画 | 口径一行 | KPI 四格、嵌套卡 |
| `/report` | 卷目 | 卷封 meta | 全章展开 |

### 4.4 读法三层（文案结构，非 UI 色块）

| 层 | 名称 | UI 处理 |
|----|------|---------|
| L1 | 格物 | 事实/盘面 — 默认展开 |
| L2 | 引经 | 典籍 — 铜左线 3px |
| L3 | 余论 | 推断 — 淡墨 + 卷末脚注 |

## 五、组件级章法

| 组件 | 规则 |
|------|------|
| **卷目登记表** | 三列：卷次｜卷名｜批语；行界分隔；无卡片阴影 |
| **六柱表** | 界画直线；五行色仅字形 |
| **Trust** | 纸底 + 朱/铜 **左边线 3px**；禁止绿黄铺底 |
| **CTA** | 全站 **1 个** 铜色主按钮/页；其余文字链 |
| **KPI** | 不在品牌首页；档案页 ≤4 格、数字铜色 |
| **顶栏** | LOGO + 篇题；导航无 active 高亮于首页 |

## 六、反模式清单（AI 国风 / 游戏 HUD）

| 禁止 | 原因 |
|------|------|
| 米黄 `#f4e8d0` 满屏 + 窗格底 | AI 模板识别色 |
| 山水、祥云、水墨大图背景 | 抢盘面；与简约冲突 |
| 朱印/铜渐变/角花/印章水印 | 装饰 > 内容 |
| 卷轴展开/盖印动效 | 游戏化 |
| 竖排堆字 + 左栏品牌柱 | 信息打架 |
| Dashboard 首屏（KPI+表+卡） | SaaS 后台感 |
| 汝窑天青作主色 | 与暖纸冲突 → 文创瓷风 |
| 黑底 LOGO 整页 | 与纸墨两境冲突（旧版问题） |

## 七、参考学习路径（团队用）

```text
Day 0  读本文 + MINIMAL-GUOFENG + LOGO v2
Day 1  打开识典古籍：记录纸色层级、目录节奏（30min）
Day 2  对照 mockups/05-home-song-target.html：首屏预算（30min）
Day 3  读 SONG-AESTHETICS-REFERENCES §2 刻本 + §4 反参考（1h）
Day 4  改一屏：只动 Token/间距/行界，不加装饰（实战）
```

**每次改 UI 前回答三问：**

1. 首屏 **一器** 是什么？  
2. 留白是 **结构** 还是 **没做完**？  
3. 有没有 **第二套配色** 在打架？
