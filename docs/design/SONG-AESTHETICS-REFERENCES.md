## 1. 宋式美学核心（文献共识）

| 关键词 | 含义 | 对 UI 的启示 |
|--------|------|--------------|
| **雅、淡、简、韵** | 宋代美学总括 | 少色、少框、少动效；韵味来自留白与校勘 |
| **格物致知** | 穷理尽性、探究秩序 | → Trust 脚注、双轨朱批、可核对盘面 |
| **以白当黑** | 留白即结构 | → 宽屏 gutter、章间天头地脚，非背景图 |
| **方、圆、素、质** | 器形单纯（宋瓷） | → 直线宫格、两档卡片，禁云纹底 |
| **平淡天真** | 水墨不重炫色 | → 墨/纸/雾为主，铜朱点缀 |
| **实用与审美并重** | 宋刻本、汝瓷共性 | → 命盘可读优先于氛围 |

**理学影响**：重内在秩序、轻外在奢华 — 与浮生「克制、可信」一致。

百科与综述：[宋代美学 - 百度百科](https://baike.baidu.com/item/%E5%AE%8B%E4%BB%A3%E7%BE%8E%E5%AD%A6/67390408) · [格物致知与宋式理性 - 设计知外](https://www.ucppt.com/culture/9521)

## 2. P0 参考 — 与浮生方案强相关（必看）

### 2.1 宋刻本版式（版面纪律）

| 资源 | 链接 | 看什么 | 浮生可迁移 |
|------|------|--------|------------|
| **宋刻本 7∶10 版框比例研究** | [雅昌艺术网摘要](https://m-news.artron.net/news/20240722/n1454727.html) | 版框高:宽≈0.65–0.75；规模化「比率」而非绝对尺寸 | 主内容区比例克制；盘面区横向展开 |
| **《宋刻本版面范式设计研究》** | [豆瓣书目](https://book.douban.com/subject/36059373/) | 版式几何、模块网格、设计规制 | 间距律 24/32px；禁止随意栅格 |
| **印刷博物馆 · 宋版结构** | [刻本版面要素](http://www.kepu.net.cn/gb/civilization/printing/evolve/evl316_02.html) | 边框、中缝、行线、双行小字注、版心 | 柱表行界、Trust 双行注、版心 meta |
| **故宫 · 皕宋特展「传媒转型」** | [册页与蝴蝶装](https://theme.npm.edu.tw/exh114/TwoHundredTreasures/ch/page-5.html) | 卷轴→方册；蝴蝶装对折；校勘严谨、纸墨古雅 | **个人档案=一编册页**；阅读方式从「刷卡片」→「翻册」 |

**版心要素对照表**

| 宋刻本 | 浮生 UI |
|--------|---------|
| 天头（书名、卷次） | 顶栏篇题 + 口径眉批 |
| 版框双边 | 主区左右册线 |
| 行界 | 六柱表 / 方盘宫格竖线 |
| 双行小字注 | 藏干、校勘脚注 11–12px |
| 鱼尾 / 折页标记 | 章节分隔细线（非色块） |

### 2.3 界画与宫格精度

| 资源 | 说明 | 浮生映射 |
|------|------|----------|
| 宋代界画（建筑人物线画） | 直线、比例、界尺 | 紫微方盘、八字柱表应用 **直线界**，不用 glow |
| 《营造法式》彩画逻辑 | 层级化用色、符号有主次 | 取 **层级逻辑**，不取青绿大面积 |

## 6. 参考 → 方案修改建议（决策表）

读完 P0 后，可在方案里做如下 **定向调整**：

| 你的担心 | 参考指向 | 建议修改方案 |
|----------|----------|--------------|
| **配色难看** | 识典古籍 + 宋刻本纸墨 | 冻结 §2.8.6 五色；**删除** Tailwind 绿黄粉 Trust 底；**取消**铜渐变按钮 |
| **布局乱** | 皕宋册页 + 7∶10 版框 | 强化「一屏一锚」；去掉 PageHead 与壳重复；母版 A 默认单栏移动 |
| **AI 化** | 融创减法 + 识典克制 | 衬线仅篇题；禁止卷轴/印章/水墨动效；用户文案不用「册线/校勘层」 |
| **不够宋** | 故宫皕宋 + 行界 | 加强 **行界**（表格栅线），而非加云纹、纸纹加重 |
| **太冷工具** | 汝瓷「温润」+ 纸感 | 保留暖纸，但 **不要** 天青大面积；用 **雾棕** 代替冷灰辅助字 |

### 6.1 建议做 A/B 的两处（改方案前可定）

| 选项 | A（当前方案） | B（参考驱动） | 参考依据 |
|------|---------------|---------------|----------|
| **主强调** | 铜金 `#b8894d` | 铜金仅 CTA；提要数字用 **墨** 加粗 | 宋刻本标题多为墨，铜用于钤印 |
| **纸面层级** | paper + surface + surface-2 三层 | paper + **白内容区** 两层 | 识典古籍、皕宋「纸墨古雅」两层即可 |

建议在 `skin-preview.html` 做 A/B 两屏并排，**你点头后再改** `variables.css`。
