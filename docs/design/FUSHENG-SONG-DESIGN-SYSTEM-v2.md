## 二、核心语法（冻结）

| 原则 | 定义 | 界面 |
|------|------|------|
| **秩序** | 8pt 网格；对齐唯一真源 | 所有间距 ∈ {8,16,24,32,48,64,96,128} |
| **比例** | 内容约 40%，空气约 60% | 首屏 ≤4 信息块 |
| **克制** | 全屏 ≤4 色相 | 见 §三 |
| **留白** | 留白 = 结构，非空边 | 用 **垂直呼吸** 分段，非窄柱+死横向留白 |
| **呼吸感** | 标题 → 空 → 一句 → 空 → 一器 → 空 → 正文 | 见 §五 |
| **器** | 每屏最多 **一件器物** | 首页 = **六卷册**（品牌结构）；档案页 = 四柱 |

## 三、配色 · 宋瓷色（冷，非米黄）

灵感：汝窑雨过天青、官窑粉青、哥窑灰青、定窑牙白。

| 角色 | Token | Hex | 隐喻 | 用途 |
|------|-------|-----|------|------|
| 天青 | `--fs-celadon-sky` | `#7a9690` | 汝窑 | 强调字、当前卷、链接 hover |
| 粉青 | `--fs-celadon-mist` | `#a8bab4` | 官窑 | 辅线、üstü eyebrow |
| 灰青 | `--fs-celadon-ash` | `#e4eae7` | 哥窑 | 页面底、分隔 |
| 牙白 | `--fs-porcelain` | `#f6f7f5` | 定窑 | 内容区、卡片底 |
| 墨 | `--fs-ink` | `#2a302e` | 松烟（偏冷） | 标题、正文 |
| 淡墨 | `--fs-ink-mist` | `#5f6b67` | 退笔 | 辅文 |
| 极淡 | `--fs-ink-faint` | `#95a09c` | 占位 | 卷次、脚注 |
| 朱 | `--fs-cinnabar` | `#8b4a42` | 朱批（冷调） | 缺失 **一行** 左线，无粉底 |

**五行色** `--wx-*`：仅干支字形，不作 UI chrome。

**废弃 v1**：`--brand-paper #f5f0e6`、`--brand-gold` 作主色 — 仅保留为 legacy 直至 Token 迁移。

## 六、线条

| 类型 | 规格 | 用途 |
|------|------|------|
| 发丝 | 1px `--fs-celadon-mist` opacity 0.6 | 器底单线 |
| 界 | 1px `--fs-celadon-ash` | 柱间界画（仅命盘） |
| 章 | **不用线** — 用 96px 间距 | 段落分节 |
| 朱批 | 3px left `--fs-cinnabar` | 缺失单行 |

**禁止**：列表每行横线、双线版框套娃、渐变册线。

## 七、组件（极简）

### 7.1 按钮

| 类型 | 样式 |
|------|------|
| 主 | 无框；天青字 + 下划线；hover 加深 |
| 次 | 1px 天青框；牙白底；小 padding |
| **禁止** | 实心铜块、gradient、圆角 pill |

### 7.2 卡片

**默认无卡片。** 仅当必须有容器：牙白底 + 1px 灰青框，**无 shadow**。

全站 shadow ≤0（书房没有浮空卡片）。

### 7.3 导航

天头 48px；品牌 + 登录；**无**五步流高亮（封面不在流内）。

## 八、首页布局（v2.2）

### 8.1 结构：左轨 + 右卷

```text
┌────────┬──────────────────────────────────────┐
│ 浮     │  宋代文人的数字书房                    │
│ 生     │  一册可核对之命书…                     │
│        │                                        │
│浮生若梦│  卷一  命之根  批 · …                  │
│        │    卷二  业之象  批 · …    ← 阶梯展开 │
│坐穷泉壑│      卷三  …                           │
│不下堂筵│        …                               │
│格物致知│          卷六  问书                    │
│        │                                        │
│六卷辑录│       进入辑录 →                       │
└────────┴──────────────────────────────────────┘
```

- **左轨**（≈148px）：竖排 `浮生` · `浮生若梦` · 竖排谚语 · 天头地脚
- **右区**：六卷 **margin-left 阶梯递增**（52px/级），卷宗竖线 + 横枝点
- **简约 ≠ 简单**：靠网格、竖排、展开节奏，不靠堆字

### 8.2 响应式

- `<768px`：左轨改横排置顶；六卷取消阶梯，单列

## 九、分页面 · 首页（数字书房）

### 9.1 用户进入

不是网页列表 → 是 **品牌书房首屏**：

1. 看见 **浮生**（品牌名题）  
2. 看见 **六卷辑录**（品牌之物，非个人四柱）  
3. 一个去向（进入辑录 / 录入）  
4. **个人档案** — 仅滚动后「案头有一册」次要块（回访用户）

### 9.2 首屏预算（品牌首页）

| 块 | 允许 |
|----|------|
| 眉批 | 1 行（数字书房） |
| 品牌名题 | **浮生** |
| 副题 | 人生六卷辑录 |
| 一句 | 品牌读法 |
| 器 | **六卷** 三行摘要 |
| 动作 | 进入辑录 + 录入提示 |
| **禁止首屏** | 用户姓名、四柱、KPI、档案 CTA |

### 9.3 滚动后（第二呼吸）

- 卷首：品牌定位 **一句话**  
- 分层：事实 / 典籍 / 推断 各一句  
- **案头**（可选）：刘博 · 命书 · 继续阅读 — 回访入口，字号小于品牌  
- 校勘：一行脚注  

### 9.4 与 v1 mockup 差异

| v1 问题 | v2 |
|---------|-----|
| 信息太满 | 首屏 4 块 |
| 米黄暖纸 | 灰青冷瓷 |
| 无节奏 | 128/64/96 垂直节律 |
| 无器 | 六卷册为品牌锚；四柱在档案页 |
| 档案当首页 | 品牌门面 + 案头回访分层 |
| 线条滥用 | 间距代线 |
