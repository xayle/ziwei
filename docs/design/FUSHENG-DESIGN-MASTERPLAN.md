## 〇、设计定案（一句话）

> **浮生是一本可以核对的人生命盘辑录：纸墨为底，界画为骨，铜朱点睛；卷目清晰，盘面最大，校勘最小。**

用户不应感到在「命理 App」或「SaaS 后台」里找数据，而应感到在 **一册私人辑录** 里按卷阅读。

## 一、设计感：情绪与视觉性格

### 1.1 情绪板（Mood Board）

```text
┌─────────────────────────────────────────────────────────┐
│  安静    像午后在书案前翻册，不是直播间算命            │
│  可信    神秘来自校勘与双轨，不来自云雾光晕            │
│  克制    全屏最多 4 种色相：纸、墨、铜、朱              │
│  精确    界画直线、占格稳定、数字对齐                    │
└─────────────────────────────────────────────────────────┘
```

### 1.2 视觉性格三要素

| 要素 | 定义 | 用户感受 |
|------|------|----------|
| **载体** | 册页 / 辑录 / 卷目 | 「这是我的命盘书」 |
| **线条** | 界画式 1px 直线宫格 | 「盘面可核对」 |
| **色** | 纸墨为主，铜朱点睛 | 「温而不俗，雅而不冷」 |

### 1.3 明确拒绝（反设计库）

- 紫红渐变、金箔玻璃、水墨大山底  
- 米白 + Noto Serif + Inter + 铜渐变按钮（AI 国风模板）  
- Tailwind 绿/黄/粉 alert 铺底  
- 圆角 16px 白卡堆叠 + 每张轻阴影  
- 顶栏 slogan + PageHead + 色块分区三重标题

## 二、色彩系统（冻结 · 不得擅自增色）

### 2.1 主盘五色（全站 chrome 只允许这些）

| 角色 | Token | Hex | 宋式隐喻 | 允许用途 |
|------|-------|-----|----------|----------|
| **纸** | `--brand-paper` | `#f5f0e6` | 熟宣纸 | `body`、shell 天头地脚外缘 |
| **内容白** | `--surface` | `#fffaf5` | 单页纸 | 卡片、盘面垫底、报告正文区 |
| **墨** | `--brand-ink` | `#1a1410` | 松烟墨 | 正文、表内主信息 |
| **雾** | `--brand-mist` | `#6b5d4f` | 淡墨注 | 眉批、辅助、表头 |
| **淡注** | `--text-3` | `#9a8b7a` | 退笔 | 占位、缺失提示 |
| **铜** | `--brand-gold` | `#b8894d` | 鎏金点缀 | KPI 数字、唯一主 CTA、active 导航、典籍左框 |
| **铜深** | `--brand-gold-dark` | `#8b5e34` | 钤印深 | KPI、hover、徽章描边 |
| **朱** | `--brand-cinnabar` | `#8b3a2a` | 朱批 | 缺失/双轨 **单行** 左线 |
| **界** | `--border` / `--border-md` | `#e5dcc8` / `#d4c4a8` | 版框/行界 | 宫格、表线、册线 |

### 2.2 纸面层级（硬性两级）

```text
全页只允许：
  纸底 #f5f0e6  →  内容白 #fffaf5
禁止：surface 套 surface-2 再套卡片的三层暖色嵌套
```

`--surface-2` **仅**用于表头行、侧栏 KPI 底 — 不得与 `--surface` 卡片再嵌套。

### 2.3 五行色（仅命盘字符）

| Token | 用途 |
|-------|------|
| `--wx-wood` … `--wx-water` | **仅**干支天干地支字形 `color` |
| 禁止 | 五行色作按钮、section 背景、导航高亮 |

### 2.4 铜色 / 朱色预算（每屏）

| 色 | 预算 |
|----|------|
| 铜 | 面积 < 8%；最多：1 CTA + active 导航 + 提要 KPI + 1 条典籍左边框 |
| 朱 | ≤ 3 处；仅左线 3px + 墨/朱字，**无粉红底** |
| 阴影 | 每页 ≤ 1 处（封面卷封 Hero） |

### 2.5 语义层色（内容类型 · 不用铺底）

典籍 / 引擎 / 推断 **不用** 绿黄粉底区分，改用 **左边框 + 小徽章**：

| 类型 | 用户文案 | 视觉 |
|------|----------|------|
| 典籍 | 典籍 | 铜章线 3px 左 + 衬线引文 16px |
| 事实 | 事实 | 无框，墨字 14px |
| 推断 | 推断 | 雾棕徽章 11px；**默认折叠** |

**删除生产用法**：`--trust-ok-bg` 绿底、`--trust-drift-bg` 黄底、`--layer-heuristic-bg` 大面积铺色。

### 2.6 按钮与交互色

| 类型 | 样式 |
|------|------|
| 主按钮 | 实心铜 `#b8894d`，字 `#fffaf5`，**禁止** linear-gradient |
| 次按钮 | 墨字 + 铜框 1px，底 `--surface` |
| 禁用 | 雾棕字 + `--border`，opacity 0.5 |
| 链接 | 铜深 `#8b5e34`，hover 下划线 |

## 三、字体系统（冻结）

### 3.1 两套字 · 分工不可混

| Token | 栈 | 气质 | 白名单 |
|-------|-----|------|--------|
| `--font-display` | `"LXGW Neo ZhiSong", "Source Han Serif SC", STSong, serif` | 刻本宋、占格稳 | 卷名、篇题、**干支盘面**、典籍引文 |
| `--font-ui` | `-apple-system, "PingFang SC", "Microsoft YaHei UI", "Segoe UI", sans-serif` | 屏读冷静 | 导航、按钮、表单、表头、脚注、KPI 标签 |
| `--font-mono` | `"JetBrains Mono", Consolas, monospace` | 等宽 | 版本号、口径代码 |

**禁止**：Inter / Roboto 作主 UI 字；第三套「国风」字体；全页 h2 衬线。

### 3.2 字号阶梯（8 档 · 不得超）

| 角色 | 字体 | 大小 | 行高 | 字重 |
|------|------|------|------|------|
| 卷名 | display | clamp(28px, 3.5vw, 36px) | 1.25 | 600 |
| 篇题 | display | 22px | 1.3 | 600 |
| 盘面干支 | display | 30px | 1.1 | 700 |
| 典籍引文 | display | 16px | 1.65 | 400 |
| 正文 | ui | 14px | 22px | 400 |
| 表头/标签 | ui | 12px | 1.4 | 500 |
| 眉批/口径 | ui | 11–12px | 1.45 | 400 |
| KPI 数字 | display 或 ui | 20px | 1.2 | 600 |

- 根字号 `html { font-size: 16px }`，正文 `0.875rem`  
- 盘面与 KPI：`font-variant-numeric: tabular-nums`  
- 禁止中文斜体；禁止 12px 以下用铜色  

### 3.3 Web Font 加载

- 默认：**零** Web Font（系统无衬线）  
- 盘面页 / 报告：自托管 **霞鹜新致宋** 子集 WOFF2，`font-display: swap`  
- 样张：`skin-preview.html` 为视觉真源，改版先改样张再改工程

## 四、布局语法（全站复用的图形语言）

### 4.1 版心常数

| 常数 | 值 | 用途 |
|------|-----|------|
| 顶栏高 | 56px | 天头 |
| 主内容宽 | 1120px | 工作台、八字、紫微 |
| 报告宽 | 1280px | 连续阅读 |
| 卷目栏 | 240px sticky | 报告左栏 |
| 正文行宽 | 62ch / 72ch | 工作台 / 报告 |
| 章间距 | 24px 或 32px 两档 | 禁止 12/16/20/28 混用 |
| 栅格 gap | 14px | 双栏母版 |

### 4.2 分隔体系（Codex Rules）

| 名称 | 规格 | 场景 |
|------|------|------|
| **界线** | 1px `--border-md` | 柱表、方盘宫格 |
| **册线** | 1px `--border` + 24px 上间距 | 区块间，替代彩色框 |
| **朱批线** | 3px `--brand-cinnabar` 左 | 缺失/双轨单行 |
| **铜章线** | 3px `--brand-gold` 左 | 典籍断语 |
| **校勘线** | 1px dashed `--border-md` | Trust 表内 |

### 4.3 卡片两档

| 档位 | 类名 | 阴影 | 每页上限 |
|------|------|------|----------|
| 平页 | `.fs-card--flat` | 无 | 不限 |
| 卷封 | `.fs-card--elevated` | 轻 | **1** |

### 4.4 一屏一锚（硬性）

每屏只允许 **一个** 视觉权重 100% 的元素：

| 页面 | 锚点 |
|------|------|
| 首页 | 卷封（档案名 + 就绪） |
| 八字 | 六柱界画盘 |
| 紫微 | 十二宫方盘 |
| 运限 | 方盘 + 日期条 |
| 报告 | 当前卷正文首段 |

## 五、内容布局规划（全书 IA · 前端呈现）

> 后端六卷是 **内容编排逻辑**；前端呈现为 **卷目 + 工作台 + 连续阅读** 三层，不绑单一 API。

### 5.1 全书卷目（用户语言）

| 卷 | 用户见名 | 前端主战场 | 母版 |
|----|----------|------------|------|
| **封面** | （档案名） | 首页卷封 | B |
| **卷首** | 怎么读这本书 | 首页折叠卡 + 报告开篇 | B / C |
| **卷一·命之根** | 四柱 · 格局 · 用神 | `/new/bazi` 速览 | A |
| **卷二·业之象** | 合冲 · 神煞 · 双轨 | `/new/bazi` 结构 | A |
| **卷三·运之波** | 大运 · 流年 · 流月 | `/new/ziwei/timeline` + 报告卷三 | A |
| **卷四·宫之图** | 十二宫 · 格局 | `/new/ziwei` | A |
| **卷五·事之理** | 事业 · 财 · 婚 … | **仅** 报告卷五 | C |
| **卷六·问书** | 追问助手 | 报告卷六，**默认折叠** | C |
| **跋·校勘** | （无大标题） | 各卷末 1 行脚注 + 报告末表 | 脚注 |

### 5.2 导航结构（顶栏 5 项）

```text
目录（→ Report 卷目） · 档案 · 命盘（→ 八字） · 紫微 · 运限
```

- 「目录」= 全书入口，不是第七个工具  
- 扩展、登录放次要位  
- 壳篇题 = 当前卷/页名，**无** slogan 堆叠  

### 5.3 母版 A — 盘面工作台（八字 / 紫微 / 运限）

```text
┌──────────────────────────────────────────────────────────┐
│ 口径眉批 11px 单行（可点击展开）                           │
├──────────────────────────────────────────────────────────┤
│ 卷首提要 SummaryStrip · KPI 铜色数字                      │
├──────────────────────────────────────────────────────────┤
│ 深度切换 [ 速览 | 结构 | 深读 ]     ← 右对齐 segmented    │
├────────────────────────────┬─────────────────────────────┤
│ 界画盘 Hero  1.45fr        │ 侧栏 0.75fr sticky          │
│ 六柱表 / 方盘              │ 流日 · 关系（结构档展开）    │
├────────────────────────────┴─────────────────────────────┤
│ 校勘脚注 1–3 行（白底细线，非 section）                    │
└──────────────────────────────────────────────────────────┘
```

**深度与卷对应**：

| 档位 | 卷 | 可见 |
|------|-----|------|
| 速览 | 卷一 | 盘面三行（主星/天干/地支）+ KPI |
| 结构 | 卷一+二 | +藏干行、神煞、关系卡 |
| 深读 | 引擎续读 | +柱间细目；**不含** 卷五生活域 |

### 5.4 母版 B — 封面与档案

**首页（封面 + 卷首）**

```text
┌─ 卷封 elevated（全站唯一阴影）────────────────────────┐
│  [Logo]  档案名 display 28–36px                        │
│  生辰一行 雾棕 12px                                      │
│  就绪朱批（单行）+ 主 CTA 铜扣                           │
│  [ 继续阅读 → 卷一 / 报告 ]                              │
└────────────────────────────────────────────────────────┘
┌─ 卷首 · 怎么读  flat ─────────────────────────────────┐
│  ① 卷一 看命之根（格局用神）                            │
│  ② 卷三 看运之波（今年流年）                            │
│  ③ 卷五 看事之理（推断需点开）                          │
└────────────────────────────────────────────────────────┘
```

**档案页**：左 65% 册页表单 · 右 35% 辑录 KPI（不重复表单文案）

### 5.5 母版 C — 报告连续阅读

```text
┌─────────┬──────────────────────────────────────────────┐
│ 卷目    │  卷名 display 22px                           │
│ 240px   │  ─── 册线 ───                                │
│ sticky  │  正文 14px / 72ch                            │
│         │  · 事实段：墨字                              │
│ 封面    │  · 典籍段：铜章线左 + display 16px           │
│ 卷一    │  · 推断段：徽章 + 手风琴默认收               │
│ 卷二    │  ─── 册线 32px ───                           │
│ …       │  卷末校勘脚注 11px 一行                        │
│ 跋      │                                              │
└─────────┴──────────────────────────────────────────────┘
```

卷目项样式：

- 当前卷：墨字加粗 + 铜点 4px  
- 未读卷：雾棕  
- 锁定卷（后期）：锁印图标 + 雾棕，无渐变  

### 5.6 卷五 · 事之理（布局专规）

每域一块，**固定四行结构**：

```text
┌─ 事业 ─────────────────────────────────────┐
│ 事实 · 引擎得分 72 · 主导十神 正官          │  ← 墨字 14px 默认展开
│ 典籍 · 「…」子平真诠 §12                    │  ← 铜章线左 display 16px
│ 推断 · 今年宜守不宜攻 …                     │  ← 雾棕徽章，默认折叠
└────────────────────────────────────────────┘
```

禁止：长段 `interpretation_text` 作主视觉；禁止 8 域平铺在八字页。

### 5.7 跋 · 校勘（最小化）

**不当独立大红章。** 两种形态：

1. **卷末单行**：`校勘 · 缺失 0 · 双轨 1 · iztro 一致` — 11px 雾棕，可点展开  
2. **报告末表**：白底细线表，仅深读用户展开

## 十一、宋式当代表达（不可全复刻 · 改动方向定案）

> **原则：留宋魂，改宋形。** 宋魂 = 雅淡简韵、格物校勘、留白秩序；宋形 = 竖排、绢色天青、卷轴动效、慢读-only 等 **载体时代性**，应现代化。

### 11.1 必须保留（宋魂 · 不动）

| 宋魂 | 为何不能丢 |
|------|------------|
| 纸墨铜朱主盘 | 品牌识别与抗 AI 模板 |
| 界画直线、盘面优先 | 命盘产品本质 |
| 留白、一屏一锚 | 可读性与气质 |
| 校勘脚注、缺失明示 | 产品可信度 |
| 卷目隐喻、事实/推断分层 | 之书 IA |
| 克制装饰 | 反玄学 SaaS |

### 11.2 建议改动（宋形 → 当代表达）

| 完全复刻会不合时代 | 建议改动方向 | 浮生定案 |
|--------------------|--------------|----------|
| **竖排正文** | 屏读横排；竖排仅卷名装饰 ≤5% | 正文横排 14px；卷名可 display 一字行题 |
| **汝窑天青主色** | 天青偏冷，与暖纸冲突；文创瓷感 | **暖纸 `#f5f0e6` 主色**；天青仅链接/实验分支 |
| **绢纸黄到看不清** | 户外屏、低对比 WCAG | 纸色固定；正文墨 `#1a1410` 对比 ≥ AA |
| **刻本宋体全文** | 小字号屏读累、Web 字体重 | 衬线 **仅** 卷名/干支；UI 用系统无衬线 |
| **蝴蝶装 3D 翻页** | 慢、晕、不像工具 | 报告 **连续滚动** + 左卷目 sticky；150ms fade 即可 |
| **极致稀疏、无密度** | 命盘需核对，过空像空状态 | 盘面区 **允许 ziwei.pub 级密度**；稀疏在 chrome 与解读 |
| **纯静态无反馈** | 用户不知加载/保存/错误 | 纸色 skeleton、铜色进度、朱批单行错误 |
| **无触控尺度** | 手机为主 | 按钮/切换 **最小 44px**；底栏 56px |
| **水墨山云底纹** | 像 AI 国风 | 仅 ≤4% 纸纹噪点，无山水 |
| **印章/卷轴/窗格纹样** | 游戏 UI 感 | 零纹样；tier 用 **方印轮廓** 2px 即可 |
| **慢读-only、无速览** | 抖音时代首屏 3 秒 | **速览默认**；结构/深读渐进 |
| **朱批仅毛笔隐喻** | 需可点击、可展开 | 朱批 = 左线 3px + 可展开脚注，非图片 |
| **暗色模式做青黛** | 易变玄幻 | **首版不做暗色**；若做 = 墨底 `#1a1410` + 纸字 `#f5f0e6`，禁紫青 |
| **四艺式极慢交互** | 点茶动画等与工具无关 | 四艺气质进 **文案/卷首**，不进交互动效 |
| **全卷平等展开** | 信息过载 | 卷五推断、卷六问书 **默认折叠** |
| **校勘藏深处** | 现代用户要可信 | 跋缩短为卷末一行 + 可展开，非删除 |

### 11.3 可主动「当代化」的五条线（推荐）

**① 阅读节奏当代化**

- 宋：坐读册页、一卷到底。  
- 今：**扫读 → 结构 → 深读** 三档；报告卷目跳转；首页「继续阅读」断点。  
- 气质不变，**路径**变快。

**② 信息密度双轨**

- 宋画留白；命盘却要 **界格满而不乱**。  
- 规则：**chrome 疏、盘面密、解读疏**。盘面允许横滚、小字藏干；壳与解读保持册页留白。

**③ 色彩当代化（暖纸派，非天青派）**

- 宋瓷天青美，但 Web 上易冷、易文创店。  
- 浮生走 **熟宣纸 + 松烟墨 + 铜钤**（识典/皕宋纸墨路线），比复刻汝窑更稳、更暖、更耐看。

**④ 字体当代化（刻本精神，屏读工程）**

- 精神：占格稳、瘦劲、典籍感。  
- 工程：子集 WOFF2 + 系统无衬线；**不**全文霞鹜新致宋。

**⑤ 信任当代化（校勘产品化）**

- 宋：批注在纸角。  
- 今：双轨、iztro、missing 做成 **一行摘要 + 展开表 + 分享钩子**（「可核对」是当代差异化，不是复古负担）。

### 11.4 明确不做的「伪宋」（加了反而丑）

| 伪宋 | 原因 |
|------|------|
| 米白 + 衬线 + 铜渐变 + 云纹 | AI 国风模板 |
| 竖排导航、竖排表单 | 可用性灾难 |
| 卷轴展开动画 | 游戏感 |
| 天青大面积铺底 | 文创瓷店 |
| 回纹/窗格底 | 站酷宋韵游戏 UI |
| 全站水墨滤镜 | 损害盘面精度 |

### 11.5 改动优先级（设计迭代）

| 优先级 | 改动 | 状态 |
|--------|------|------|
| P0 | 横排 + 暖纸墨 + 系统 UI 字 | 已定案 / 部分落地 |
| P0 | 速览三档、跋脚注化 | S2 部分落地 |
| P1 | 报告滚动卷目（非翻页） | 待 D4 |
| P1 | 盘面密度上限放宽 | 待紫微/八字实测 |
| P2 | 纸色暗色模式（墨底版） | 可选 |
| P2 | 卷名竖排装饰（首页/报告封面 only） | 可选 ≤5% |
| P3 | 天青链接色 A/B | 实验 |

**结论**：浮生应是 **「宋式典籍册页的当代转译」**，不是宋代界面复刻。改的是载体与节奏，不改的是雅淡简韵与校勘可信。
