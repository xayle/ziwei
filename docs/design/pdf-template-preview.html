出生高置信

本命年当前 / 下一次

优先提示本命年、换运、冲克与关键年份，作为人工讲解前置提醒。

换运临界期

把大运切换点与敏感年份提前标记，减少解释时的跳转成本。

冲克 / 校验提醒

将冲克与结果校验并列展示，避免用户只看到结论，看不到依据。
