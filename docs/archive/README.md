# 归档目录

本目录存放**不应作为当前开发事实来源**的文档。读代码和做决策时，请优先使用 [`../README.md`](../README.md) 列出的活跃文档。

## 子目录

| 目录 | 用途 | 是否可读 |
|------|------|----------|
| [`superseded/`](superseded/) | 已被新文档取代、或描述已删除路由/脚本的材料 | 仅回溯对照 |
| [`drafts/`](drafts/) | txt 草稿、外部路径笔记、一次性迁移清单 | 一般忽略 |
| [`deliverables/`](deliverables/) | Word 报告样例 | 产品参考，非技术文档 |
| [`history/`](history/) | 2026-02~04 周报、阶段验收、缺陷修复记录 | 历史审计 |
| 根目录 `*.md` | 旧版部署/运维指南 | 优先用 `../guides/DEPLOYMENT-GUIDE.md` |

完整分类说明见 [`../DOCS-AUDIT.md`](../DOCS-AUDIT.md)。
