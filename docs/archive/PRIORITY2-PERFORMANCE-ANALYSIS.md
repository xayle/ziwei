## 🔧 监控指标配置

### Prometheus 告警规则更新

添加以下告警规则到 `alerts.yml`:

```yaml
groups:
  - name: bazi_performance
    interval: 15s
    rules:
      - alert: HighLatencyP95
        expr: histogram_quantile(0.95, http_request_duration_seconds_bucket) > 0.15
        for: 5m
        annotations:
          summary: "P95 延迟超过 150ms"
          description: "当前 P95: {{ $value }}s"

      - alert: LowThroughput
        expr: rate(http_requests_total[1m]) < 400
        for: 5m
        annotations:
          summary: "吞吐量低于 400 req/s"
          description: "当前吞吐量: {{ $value }} req/s"

      - alert: HighErrorRate
        expr: rate(http_requests_total{status=~"5.."}[5m]) > 0.01
        for: 2m
        annotations:
          summary: "错误率超过 1%"
          description: "错误率: {{ $value }}"
```

## 🎓 性能测试方法论

### 本次测试使用的工具

- **脚本**: `performance_benchmark.py`

1. 本地网络测试，无网络延迟
2. 未包含数据库实际查询
3. 未测试大 payload 响应
