## 0. 本文解决什么

在「暖色宣纸 + 铜金」的模糊描述之上，**选定一种可命名、可引用、可验收的艺术风格**，并给出参考源、视觉语法与落地规则。  
工程、美术、界面规划均按本文执行；新增装饰不得脱离定案风格。

## 1. 候选风格评估（为何不是别的）


| 风格             | 代表参考              | 与浮生匹配度                   | 结论                    |
| -------------- | ----------------- | ------------------------ | --------------------- |
| **宋式典籍册页极简** ★ | 识典古籍、宋代版刻、界画      | 命盘=册页；Trust=校勘脚注；分层断语=经注 | **采用**                |
| 极简新中式（消费向）     | 观夏、无印             | 留白与低饱和好；缺「文献/校勘」语义       | **吸收留白与材质，不作主风格**     |
| 海派 Art Deco 摩登 | 上海摩登体、天机玄鉴        | 装饰线、几何字过强；偏海报与营销         | **仅标题装饰可参考 5%，禁作主视觉** |
| 园林水墨意境         | 水墨晕染、山水留白         | 氛围好；损害盘面精度与数据可读          | **拒绝**                |
| 玄学金箔玻璃         | ZiweiKnows 分享卡    | 与 PRODUCT 反例完全一致         | **拒绝**                |
| 纯工具扁平          | Baguame、ziwei.pub | 清晰但无品牌记忆、无文书感            | **仅借鉴交互，不作美术主调**      |
| 清宫华丽彩画         | 青绿旋子、重度纹样         | 装饰压过内容                   | **拒绝**                |




### 1.1 定案一句话

> **浮生视觉 = 宋代典籍册页的版式纪律 + 现代极简界面的留白与对比度。**

像在一册可翻阅的命盘辑录里阅读：有版心、有朱批、有校勘脚注，而不是在玄学 App 或 SaaS 后台里找数据。

## 2. 风格定义：宋式典籍册页极简



### 2.1 风格关键词


| 维度       | 定义                                |
| -------- | --------------------------------- |
| **时代气质** | 宋代——理性、克制、法度分明（《营造法式》式秩序，非富丽清宫）   |
| **载体隐喻** | 册页、辑录、手批——个人档案是一「编」而非一次性打印        |
| **线条性格** | 界画式直线：宫格、表格、分隔线清晰挺劲，拒绝圆角卡片堆叠      |
| **色彩性格** | 纸、墨、铜、朱——低饱和、低装饰；五行色 **仅** 用于干支字符 |
| **神秘来源** | 来自 **校勘与双轨**，不来自云雾、光晕、星座插画        |




### 2.2 情绪板（Mood）

```text
安静 │ 可核对 │ 纸页摩擦感 │ 细线框 │ 偶有点状朱批
──────────────────────────────────────────────────
不要：金箔 │ 紫雾 │ 玻璃 │ 霓虹 │ 水墨大山 │ 八卦背景
```



### 2.3 与品牌 slogan 的映射


| Slogan | 视觉表达                    |
| ------ | ----------------------- |
| 浮生若寄   | 留白多、人生阶段用时间轴/卷次，不用戏剧性插画 |
| 知命     | 盘面 Hero 最大权重            |
| 知心     | 解读区衬线长文、卷末式折叠           |

## 3. 参考源体系（按优先级）



### 3.1 一级参考 — 必须对齐


| 参考             | 类型    | URL / 来源                                                                           | 学什么                 | 不学什么       |
| -------------- | ----- | ---------------------------------------------------------------------------------- | ------------------- | ---------- |
| **识典古籍**       | 数字阅读  | [https://www.shidianguji.com](https://www.shidianguji.com)                         | 横排阅读、目录+正文、降低门槛的文献感 | 不做竖排切换为主   |
| **永乐大典影像库**    | 博物馆数字 | [https://kvlab.org/project/yongledadian/](https://kvlab.org/project/yongledadian/) | 满版纸面、文物级纸色、结构化导览    | 不过度 3D 翻页  |
| **宋代界画**       | 艺术史   | 界画（建筑人物线画）精度                                                                       | **宫格/柱表直线体系**、比例严谨  | 不复制建筑纹样    |
| **《营造法式》彩画逻辑** | 典籍    | 学术/博物馆展陈                                                                           | 主次色阶、符号有层级          | 不引入青绿大面积铺色 |




### 3.2 二级参考 — 提炼原则


| 参考                    | 学什么                      |
| --------------------- | ------------------------ |
| **观夏**（To Summer）     | 极简新中式留白、单点暖色、包装上的细线框     |
| **无印良品 MUJI**         | 材质诚实、无多余装饰、标签 typography |
| **Baguame**           | 盘面优先、渐进披露（功能，非皮肤）        |
| **ziwei.pub / iztro** | 方盘字号分区、运限线框（功能，非皮肤）      |




### 3.3 三级参考 — 限制使用 ≤5%


| 参考                      | 可用范围                                    |
| ----------------------- | --------------------------------------- |
| **海派 Art Deco / 上海摩登体** | 仅 **首页档案名 Display 字** 可带轻微几何装饰线；导航/表单禁用 |
| **PMAI 八字原型**           | 仅 **录入→结果旅程** 信息顺序                      |




### 3.4 反参考清单（禁止库）

- ZiweiKnows 黑金箔卡、玻璃拟态背景  
- 紫红渐变 + 星座粒子  
- Element Plus 默认表格蓝灰  
- 大面积 Trust 粉红 alert section  
- 云纹、回纹、八卦水印铺底

## 4. 视觉语法（Style Grammar）

以下为浮生 **独有** 的图形语言，所有页面复用。

### 4.1 版心（Text Column）


| 项    | 规则                             |
| ---- | ------------------------------ |
| 最大行宽 | 正文 `62ch`；报告 `72ch`            |
| 天头地脚 | 页顶 `24–32px` 呼吸后再进内容；章末 `40px` |
| 栏距   | 双栏 gap `28px`；目录栏 `240px` 固定   |




### 4.2 分隔（Codex Rules）


| 元素      | 规格                                    | 用途                   |
| ------- | ------------------------------------- | -------------------- |
| **界线**  | `1px solid var(--border-md)`          | 六柱表、方盘宫格             |
| **册线**  | `1px solid var(--border)` + `8px` 上间距 | 区块之间，**替代彩色大框**      |
| **朱批线** | `3px solid var(--brand-cinnabar)` 左侧  | 仅缺失/双轨 **单行** 提示，非整段 |
| **铜章线** | `3px solid var(--brand-gold)` 左侧      | 典籍断语块                |
| **校勘线** | `1px dashed var(--border-md)`         | Trust 表内行分隔          |




### 4.3 朱批与脚注（Annotation System）


| 层级       | 视觉                  | 文案气质                   |
| -------- | ------------------- | ---------------------- |
| **朱批**   | 11px 朱色 + `!` 图标；单行 | 「缺失」「双轨」               |
| **校勘脚注** | 12px 墨色；白底细框表       | Trust、iztro、provenance |
| **眉批**   | 11px 雾棕；口径 banner   | 子时规则、真太阳时              |


Trust 区 = **页脚校勘记**，不是 alert banner。

### 4.4 印章（Seal）— 极克制


| 可用                                           | 禁用          |
| -------------------------------------------- | ----------- |
| tier **全书口径** 徽章可做 **方印轮廓**（圆角 2px 方形，非圆形八卦） | 登录页大印章、背景水印 |
| 每页最多 **1 枚** 视觉印章感                           | 动效盖章        |




### 4.5 纸与墨（Surface）

```css
/* 定案 — 写入 variables.css */
--brand-paper: #f5f0e6;   /* 熟宣纸 */
--brand-ink:   #1a1410;   /* 松烟墨 */
--surface:     #fffaf5;   /* 单页纸 */
--paper-grain: opacity ≤ 4% 噪点 SVG，仅 body/shell */
```

阴影：几乎不用；**边框优先于阴影**（与 Minimal UI 原则一致）。

### 4.6 字体阶梯（Typography Scale）


| 角色     | 字体                   | 字号                     | 备注         |
| ------ | -------------------- | ---------------------- | ---------- |
| **卷名** | Noto Serif SC        | clamp(28px, 4vw, 40px) | 首页档案名、报告卷首 |
| **篇题** | Noto Serif SC        | 22–26px                | 页标题        |
| **条目** | Noto Serif SC        | 17–18px                | 卡片 h2      |
| **正文** | Inter / Noto Sans SC | 14–15px                | UI、表       |
| **眉批** | Inter                | 12–13px                | 口径、meta    |
| **校勘** | Inter                | 11–12px                | Trust 脚注   |
| **干支** | Noto Serif SC        | 28–32px                | 盘面字符，五行色   |


**Web Font**：生产环境加载 `Noto Serif SC` + `Inter`（与 `skin-preview.html` 一致）。

### 4.7 动效（Motion）


| 允许                          | 禁止          |
| --------------------------- | ----------- |
| 运限切换 150ms fade             | 水墨扩散、翻页 3D  |
| 手风琴 200ms                   | 粒子、光晕 pulse |
| `prefers-reduced-motion` 归零 | 循环背景动画      |

## 5. 色彩定案（从宋式提炼，非复制青绿）

宋代建筑彩画以 **理性分层** 著称；浮生取其 **层级逻辑**，不取其色相：


| 浮生角色 | Token              | 宋式来源隐喻             |
| ---- | ------------------ | ------------------ |
| 纸    | `--brand-paper`    | 绢纸底                |
| 墨    | `--brand-ink`      | 松烟墨字               |
| 铜    | `--brand-gold`     | 铜鎏金 **点缀**（非满涂）    |
| 朱    | `--brand-cinnabar` | 朱批圈点               |
| 雾    | `--brand-mist`     | 淡墨注释               |
| 五行   | `--wx-*`           | **仅干支字符**，不作 UI 底色 |


**禁用**：青绿旋子大面积、金碧山水底、科技蓝紫渐变。

## 6. 分场景应用（完整方案）



### 6.1 应用壳 `NewAppShell`


| 元素     | 宋式册页表达                         |
| ------ | ------------------------------ |
| 顶栏     | **天头**：56px；底边 `1px` 册线；无渐变    |
| Logo 区 | 不叠 slogan；页名 = **篇题** 字号       |
| 导航     | 5 步；active = 铜色下划线 2px（类似卷目索引） |
| 背景     | 熟纸 + 4% 纸纹；**禁止** 蓝晕           |




### 6.2 首页 / 档案（母版 B）


| 元素     | 表达                               |
| ------ | -------------------------------- |
| Hero   | 唯一 elevated：**卷封**感（轻阴影、卷名、就绪朱批） |
| 表单     | 册页条目；label 上置；**无** nested 灰框    |
| 侧栏 KPI | **辑录摘要**数字；不重复表单                 |




### 6.3 八字 / 紫微（母版 A）


| 元素           | 表达                      |
| ------------ | ----------------------- |
| 六柱表          | **界画表格**：直线宫格、日主列淡金底    |
| 方盘           | 宫格界线；选中宫铜描边；无 glow      |
| SummaryStrip | **卷首提要**横条              |
| 深度切换         | 速览 / 结构 / 深读 = **卷目详略** |
| 解读           | 典籍=铜左边框；引擎=褐；启发式=淡黄+折收  |




### 6.4 报告（母版 C）


| 元素  | 表达                 |
| --- | ------------------ |
| 目录  | 左侧 **卷目索引** sticky |
| 章标题 | 衬线篇题 + 下册线         |
| 互证表 | **校勘表**样式（细线、无斑马纹） |
| 打印  | 去导航；纸白；墨黑；朱批仅缺失项   |




### 6.5 扩展 Hub


| 元素     | 表达                      |
| ------ | ----------------------- |
| 四卡     | **门类入口**：细框 + 箭头；禁插画占位  |
| 合盘（规划） | 双册并排（Baguame 布局 + 宋式册线） |
