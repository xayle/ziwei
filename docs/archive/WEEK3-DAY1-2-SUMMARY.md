## 🔑 關鍵改進

### 密碼安全
```
舊: SHA256 (1ms, 容易破解)
新: Argon2 (100ms, 1000倍更強)
```

### 令牌管理
```
舊: 僅支持 access_token (24小時)
新: access_token (24h) + refresh_token (7天)
```

### 註銷能力
```
舊: 無法真正註銷
新: 數據庫撤銷所有用戶令牌
```
