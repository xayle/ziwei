## 1. 决策锁定

用户已确认终点：

> 五书结构完整 + 关键条目大量 verified。

五书：

1. 《滴天髓》（含阐微等同宗注疏条目，以书名含「滴天髓」计）  
2. 《子平真诠》  
3. 《穷通宝鉴》  
4. 《三命通会》  
5. 《渊海子平》

原则：

- **结构齐** = 产品/引擎会打的类目有覆盖，不是 PDF 字节齐。  
- **质量齐** = 关键条 `verification_status=verified`，才可进 cite /「典籍依据」。  
- **未核条**可入库，但 **不得** 进 cite 层。  
- 他书（千里命稿、神峰、易经等）保留语料，**不进本战役终点定义**；cite 策略另见 §5。

与 [post-remediation-next-plan](./2026-07-16-post-remediation-next-plan.md) 关系：本战役优先于该文 N-08 GTM；洞察/短读（N-01～N-06）可并行，但不消耗语料人力。

## 4. 应收 / 待核清单（执行序）

### 4.1 P0 · 《穷通宝鉴》（最缺）

**结构应收：**

| 批次 | 内容 | 建议条数 | 优先 verified |
|------|------|----------|---------------|
| QT-01 | 总论 / 调候大纲 | 5～8 | 全部 |
| QT-02 | 月令寒暖矩阵（按十二月气或月支） | ≥12 | 全部 |
| QT-03 | 天干在各月令宜忌（甲～癸×代表月） | ≥40（可分期） | 先核引擎调候能打到的交叉点 |
| QT-04 | 与 `classic_refs` / `tables.py` 调候口径对齐条 | 按 diff | 冲突条必须 verified 或标 drift |

**不做：** 全文影印无条目化；无 spotcheck 的长文粘贴。

| 批次 | 书 | 内容 |

| 批次 | 书 | 内容 |

| ZP-02 | 子平 | 十神专章缺口 |

### 4.4 核验工序（每条必走）

1. 条目化入库：`id / title / passage / tags / author|dynasty`  
2. spotcheck：对照底本或 ctext；填 `docs/reports/spotcheck-ctext.md` 或同级表  
3. `verification_status=verified`  
4. 挂 `classic_refs` / explain 检索能命中（有则加 mapping 测）
