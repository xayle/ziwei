八字验证 · 使用说明

步骤
1) 填写日期时间（当地时间）、时区、经度；2) 提交后查看排盘、级别、推荐时制；3) 告警/边界提示会影响解读开启状态。
字段解释
  • 推荐：none=无、beijing_time=北京时、solar_time=真太阳时。
  • 级别：L0=无差异，L1=时柱差异，L2=月柱差异，L3=日/年柱差异。
  • 风险：near_shichen_boundary / near_jieqi_boundary 为边界风险，解读可能关闭。
操作提示
· 若推荐为空，本页已显示“无”；· 真太阳时开关在验证页左侧表单；· 返回 JSON 可点“复制”按钮。

紫微斗数 · 使用说明

步骤
1) 填写阳历生日、时辰；2) 点击"排盘"自动生成命盘；3) 查看命宫、大运、流年、飞星四化等面板。
功能说明
  • 命宫：安命 / 身宫 / 五行局 / 纳音 / 14主星布局。
  • 大运：顺/逆行，按节气精算，含流年流月。
  • 飞星四化：宫干化禄/权/科/忌，自宫飞化叠加。

常见问题 / 注释

真太阳时 vs 北京时
真太阳时仅影响时柱；推荐=真太阳时时，优先解读 solar_time；推荐=北京时 或 无 时，按常规北京时间排盘。
告警与解读
存在边界风险或 near_shichen_boundary 时，解读可能关闭，财运/婚姻/桃花/大运面板会隐藏。
鉴权失败
接口返回 401 时，请检查 token 是否在 URL、localStorage.token/access_token 或 sessionStorage.access_token 中；本地可用 LOCAL_AUTH_BYPASS=1。
